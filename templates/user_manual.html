User Manual

Welcome to the Dataset Preprocessing Tool. This manual will guide you through the various features and functionalities of the website.

Uploading a Dataset

To begin using the tool, upload a CSV file containing your dataset. Click the "Upload CSV File" button, select your file, and click "Upload".

Handling Missing Values

If your dataset contains missing values, you can handle them by selecting the column and clicking the "Handle Missing Values" button. The tool will fill missing values with the mean for numerical columns and the mode for categorical columns.

Encoding Categorical Columns

To encode categorical columns, select the column you want to encode and click the "Encode Categorical Columns" button. The tool will convert categorical values into numerical values.

Dropping Duplicates

To remove duplicate rows from your dataset, click the "Drop Duplicates" button. This will ensure that your dataset only contains unique records.

Scaling Numeric Columns

If you want to standardize the scale of numeric columns, select the column and click the "Scale Numeric Column" button. The tool will apply standard scaling to the selected column.

Deleting Columns

To remove an entire column from your dataset, select the column and click the "Delete Column" button. This will permanently remove the selected column from your dataset.

Undoing Operations

If you wish to revert the last operation performed, click the "Undo Last Operation" button. This will undo the most recent change made to the dataset.

Viewing Column Statistics

To view statistics of a specific column, select the column and click the "Show Column Statistics" button. The statistics include maximum, minimum, mean, standard deviation, variance, median, and mode.

Validate DataFrame

To ensure that your dataset meets certain quality criteria, click the "Validate DataFrame" button. This will check for issues such as missing values, data types inconsistencies, and other potential data quality problems.

Remove Outliers

To remove outliers from your dataset, select the column you wish to clean and click the "Remove Outliers" button. The tool will identify and remove values that fall outside a specified range or distribution.

Replace Values

If you need to replace specific values in your dataset, use the "Replace Values" feature. Select the column and specify the value to be replaced and its replacement. Click the "Replace Values" button to make the changes.

Delete Selected Values

To remove specific values from a column, use the "Delete Selected Values" button. Choose the column and specify the values to be deleted. This will remove only the selected values, leaving other data intact.

Viewing Dataset Details

Click the "Show Dataset Details" button to see an overview of your dataset, including the number of rows, columns, and the count of numerical and categorical columns.

Generating Plots

You can visualize your data by generating plots. Select the type of plot (Histogram, Scatter Plot, or Box Plot) and choose the relevant columns for the X and Y axes. Click "Generate Plot" to view the plot.

Exporting Data

To save the processed dataset as a CSV file, click the "Export CSV" button. You can also download the label encoders and scalers used in the preprocessing by clicking the "Export Encoders & Scalers" button.

Contact

If you have any questions or need further assistance, please contact support.
